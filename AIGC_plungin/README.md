# AI工具聚合助手

一个Chrome浏览器插件，用于聚合展示AI工具和设计文章。

## 功能特点

- 展示最新设计文章
- 聚合展示各类AI工具
- 支持工具分类浏览
- 支持模糊搜索
- 美观现代的UI设计

## 安装方法

1. 下载本项目代码
2. 打开Chrome浏览器，进入扩展程序页面（chrome://extensions/）
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择本项目文件夹

## 使用说明

1. 点击浏览器工具栏中的插件图标打开主界面
2. 在搜索框中输入关键词可以搜索工具和文章
3. 使用分类标签可以筛选不同类型的AI工具
4. 点击文章卡片或工具卡片可以跳转到相应页面

## 技术栈

- HTML5
- CSS3
- JavaScript
- Chrome Extension API

## 注意事项

- 本插件需要网络连接才能正常获取文章内容
- 部分功能可能需要相应的网站权限
